Tips:
Keep WYSIWYG turned off! it messes up the coding big time D:
 
In this i use RGBA to set colors, you can change the opacity of the boxes using the last number im the bracket. It should be kept somewhere between 0 and 1, 0 being invisible and 1 being a solid colour
 
Wherever you see URL written, insert a link! I use Imgur to insert photos, but Unsplash is also a brilliant source of free background photos. If you upload a photo to Imgur and it isn't yours (aesthetic photo etc) make sure it's free to use first and credit at the end!
     - if you struggle to make images show up, make sure the link is in image format (e.g i.imgur.com instead of imgur.com) and after the link you put .png or .jpg !
     
If there's anything you're struggling with, dont't hesitate to message me! I'm more than happy to help
 
If you're new to using HTML templates, try and only edit the URLs and grey boxes! I'll try and make this as guided as possible so hopefully you dont get stuck!
 
You can edit the little symbols using the site FontAwesome, every time you see a phrase without the quotation marks like 'fal fa-cross' it's a symbol! If you'd like to, you can change the symbols by swapping that phrase with one from a different symbol from the sitesite
